# TaxonTreeMakie — API Reference

`PaleobiologyDB.TaxonTreeMakie` is an optional package extension that renders
[`TaxonTree`](@ref) objects as Makie dendrograms.  It activates automatically
when any Makie backend (e.g. `CairoMakie`) is loaded.

See the [TaxonTreeMakie guide](../guide/taxontree_makie.md) for installation
instructions, worked examples, and a full attribute reference.

```@docs
TaxonTreeMakie
```

## Plot type

`TaxonTreePlot = Plot{taxontreeplot}` is the Makie plot type generated by the
`@recipe` macro. For the full attribute table and usage examples, see the
[TaxonTreeMakie guide](../guide/taxontree_makie.md).

## Standalone figure

```@docs
taxontreeplot
```

## Existing axis

`taxontreeplot!(ax, tree; kwargs...)` — same keyword arguments as [`taxontreeplot`](@ref).
See the [TaxonTreeMakie guide](../guide/taxontree_makie.md) for usage examples.
