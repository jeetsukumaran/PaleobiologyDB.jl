# TaxonomyTreeMakie — API Reference

`PaleobiologyDB.TaxonomyTreeMakie` is an optional package extension that renders
[`TaxonomyTree`](@ref) objects as Makie dendrograms.  It activates automatically
when any Makie backend (e.g. `CairoMakie`) is loaded.

See the [TaxonomyTreeMakie guide](../guide/taxonomytree_makie.md) for installation
instructions, worked examples, and a full attribute reference.

```@docs
TaxonomyTreeMakie
```

## Plot type

`TaxonomyTreePlot = Plot{taxonomytreeplot}` is the Makie plot type generated by the
`@recipe` macro. For the full attribute table and usage examples, see the
[TaxonomyTreeMakie guide](../guide/taxonomytree_makie.md).

## Standalone figure

```@docs
taxonomytreeplot
```

## Existing axis

`taxonomytreeplot!(ax, tree; kwargs...)` — same keyword arguments as [`taxonomytreeplot`](@ref).
See the [TaxonomyTreeMakie guide](../guide/taxonomytree_makie.md) for usage examples.
